# getlimen organization meta-repo

This is the special GitHub `.github` repo that holds organization-level templates and the public [profile README](./profile/README.md).

See [`profile/README.md`](./profile/README.md) for the organization landing page (rendered at [github.com/getlimen](https://github.com/getlimen)).

## What belongs here

- `profile/README.md` — the org landing page
- `profile/assets/` — SVG logos, banners
- Issue templates (future)
- PR templates (future)
- Code of conduct (future)

## What does NOT belong here

- Any project code (that goes in the per-component repos)
- Design specs, plans, handoffs (those live in `getlimen/limen/docs/`)
